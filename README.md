# 💰 Advanced Personal Finance Tracker (Java)

A feature-rich console-based Personal Finance Management System built using Java.
This application allows users to manage transactions, set budgets, track financial goals, receive smart notifications, and generate detailed financial reports.

---

## 🚀 Features

### 👤 User Management
- Register new account
- Login system
- Update monthly income
- Profile management

### 💳 Transaction Management
- Add Income & Expense
- View transactions (All / Category / Date Range)
- Delete transactions
- Automatic balance calculation

### 📊 Budget Tracking
- Set monthly category budgets
- Track spending percentage
- Budget exceeded alerts
- Near-limit warning (>90%)

### 🎯 Financial Goals
- Create financial goals
- Track progress
- Goal completion detection
- Overdue goal alerts
- Days remaining calculation

### 📈 Analytics & Reports
- Current balance calculation
- Monthly savings rate
- Category-wise expense breakdown
- Income vs Expense analysis
- Generate comprehensive financial report

### 🔔 Smart Notifications
- Negative balance warning
- Low balance alert
- Budget exceeded alert
- Goal overdue notification
- Goal completion congratulation

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming
- Java Collections
- File Handling (Serialization)
- Java Time API
- Streams & Lambda Expressions
- Console UI with ANSI Colors
